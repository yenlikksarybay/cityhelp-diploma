## О проекте

**CityHelp** — интеллектуальная веб-платформа для управления городскими обращениями.  
Система цифровизирует взаимодействие между жителями и муниципальными службами.

Платформа:

- объединяет граждан и городские органы управления  
- визуализирует обращения на интерактивной карте  
- автоматизирует процесс обработки заявок  
- использует методы искусственного интеллекта для классификации и приоритизации  

Проект сочетает современные веб-технологии и элементы машинного обучения и представляет собой социально значимое решение в сфере цифровизации городского управления.

---

## Схема работы системы

```
Гражданин отправляет обращение
Сервер сохраняет данные → статус «Новое»
AI-модуль классифицирует и определяет приоритет
Администратор проверяет и назначает исполнителя
Исполнитель выполняет работу → обновляет статус
Гражданин получает уведомление
Система формирует аналитику и прогнозы
```

## Основной функционал

### Гражданин
- Создание обращения
- Загрузка фотографий
- Просмотр обращений на карте
- Отслеживание статуса
- Получение уведомлений

### Администратор
- Просмотр всех обращений
- Фильтрация и управление заявками
- Назначение исполнителей
- Изменение статусов
- Доступ к аналитике

### AI-модуль
- Автоматическая классификация обращений
- Определение категории проблемы
- Оценка приоритета
- Формирование аналитических данных
